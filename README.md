## Installation

1. Clone the repository:
```bash
git clone <https://github.com/selincece/secret-wall>
cd secret-wall
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
npm start
```


## Usage

1. **Register**: Create a new account with email and password
2. **Login**: Sign in with your credentials or use Google OAuth
3. **Submit Secrets**: Share your anonymous secrets
4. **View Secrets**: See your submitted secrets
